Thndr Security CTF

This project contains my writeup and solutions for the Thndr Security CTF challenge. During the assessment, I discovered multiple vulnerabilities including SQL Injection, Command Injection, and Kubernetes Secret Enumeration. By exploiting these issues, I was able to retrieve three JWT-based flags from the environment.

The challenge demonstrated how insecure application code combined with weak Kubernetes permissions can lead to privilege escalation and secret disclosure inside containerized environments.
